# Fake Job Detection System

## Overview
This project is a Machine Learning and Flask web application that predicts whether a job posting is Real or Fake.

## Features
- Detects fake job postings
- User-friendly web interface
- Machine Learning prediction
- Fast and accurate results

## Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- HTML
- CSS

## Project Structure

fake-job-detector/
├── app.py
├── train_model.py
├── dataset/
├── model/
└── templates/


## How to Run
1. Install Python.
2. Install the required packages:

pip install flask pandas scikit-learn

3. Run:

python app.py

4. Open:

http://127.0.0.1:5000
